 Adaptive Energy Saving Room Controller based on the PIC16F877A microcontroller. The system monitors temperature, ambient light, and room occupancy to control fan speed and LED brightness using PWM. It features a people counter, SAVE mode, LCD and UART monitoring, and a Timer1 interrupt
The project is an Adaptive Energy Saving Room Controller implemented using the PIC16F877A microcontroller. Its purpose is to automatically manage room energy consumption by monitoring temperature, ambient light, and occupancy conditions.

The system controls a fan and a lighting unit using PWM signals while providing real-time monitoring through an LCD display and UART communication.

Inputs

The controller receives information from the following inputs:

1. LM35 Temperature Sensor

Measures room temperature and sends an analog voltage to the PIC ADC.

2. LDR Sensor

Measures ambient light intensity and sends an analog signal to the ADC.

3. Enter Button

Simulates a person entering the room.

4. Leave Button

Simulates a person leaving the room.

Processing

The PIC16F877A continuously acquires sensor data and executes control algorithms.

A Timer1 interrupt-driven scheduler organizes the system tasks:

Task	Period
Sensor Reading	100 ms
LCD Update	500 ms
UART Report	1 s
Timeout Check	1 s

This allows the system to operate in real time.

Temperature Control

The LM35 reading is converted to temperature using the ADC.

The controller adjusts the fan speed according to temperature:

Temperature	Fan Speed
< 25°C	0%
25–26°C	50%
≥ 27°C	100%

PWM generated by CCP1 controls the fan.

Lighting Control

The LDR measures room brightness.

The controller automatically adjusts LED brightness:

Light Level	LED Brightness
Low Light	High Brightness
Medium Light	Medium Brightness
High Light	Low Brightness

PWM generated by CCP2 controls the LED.

Occupancy Management

The system maintains a people counter.

Enter Button

People count increases.

Leave Button

People count decreases.

The counter is limited between:

0 ≤ People Count ≤ 5

This prevents overflow and invalid occupancy values.

Occupancy Indicators

Two LEDs indicate room status:

Green LED

Room occupied (people > 0)

Red LED

Room empty (people = 0)

This provides a quick visual indication of occupancy.

Energy Saving Mode

When the occupancy count becomes zero:

Inactivity timer starts.
System waits 5 seconds.
SAVE mode activates automatically.

In SAVE mode:

Fan operation is reduced or disabled.
LED brightness is reduced.
Energy consumption decreases.

When a person enters again:

SAVE mode exits.
NORMAL mode resumes immediately.
LCD Display

The 16×2 LCD displays:

Temperature
Fan Speed
Light Level
LED Brightness
People Count
Operating Mode

Example:

T:044C F:100%
P:003 L:020%
UART Monitoring

Every second the system sends a status report to the UART terminal:

TEMP=044C
LIGHT=020%
FAN=100%
LED=060%
PEOPLE=003
MODE=NORMAL

This is useful for debugging and monitoring.
